YOLOv5

A breaking edge computer vision technology. A highly reliable way to train AI. YOLOv5 is a fast and efficient convolutional neural network (CNN) to tackle computer vision-related problems.

YOLO stands for You Only Look Once

TensorFlow

An open source platform used for anything machine learning. A state-of-the-art and very useful tool. It can be especially useful for training deep neural networks.

RoboFlow

An annotation tool for training data. RoboFlow also allowed us to augment our data using methods such as rotating, shearing, and blurring each image to make sure our model extracts more information for every image.

Serp API

A web scraping tool used for collecting images for use in data sets.

Motivation

We wanted to create this AI to help law enforcement be informed about potential problems so they can accurately respond way before someone calls 911. This model will help provide a proactive method for weapon detection that could be integrated with security cameras. Using this AI will help reduce response times to deadly events, and provide information to law enforcement to keep more people safe. Thousands of people die every year due to weapons and we want to reduce that number to help save lives. This can be implemented in schools, public places (malls, parks, concerts etc.), and in large crowds to help detect potential problems before they can cause harm.

Model and Dataset

We compiled over 800 images into a dataset to train our current model. We have used Roboflow to carefully annotate every image. From there our model was trained using YOLOv5 creating a accurate and reliable model.

Future Scope

In the future we hope to collect more data and train our model to become even more accurate. We also want to experiment with different versions of YOLOv5 such as YOLOv5M or YOLOv5L once we gather a larger data set to train with more images.
